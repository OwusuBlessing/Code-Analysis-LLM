GitHub Code Analyzer

Welcome to our simple app that connects you to GitHub. Analyze code from your repositories using a Large Language Model (LLM).

To use this app, you need to set up your environment. Follow these steps:

  1. Create a .env file in the root of your project.
  2. Provide your GitHub client ID, GitHub secret key, and OpenAI key in the .env file.
  3. Here's an example of the content in the .env file:
                
                    GITHUB_CLIENT_ID=your_github_client_id
                    GITHUB_SECRET_KEY=your_github_secret_key
                    OPENAI_API_KEY=your_openai_key
